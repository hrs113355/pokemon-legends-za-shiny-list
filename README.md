# 🎮 寶可夢 ZA 色違收集清單

一個用來追蹤寶可夢 ZA 版本色違收集進度的互動式網頁工具。輕鬆管理色違收集目標，視覺化追蹤進度！

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

---

## ✨ 功能特性

### 📊 完整的收集追蹤系統
- ✅ **三種狀態管理**：已收集、未收集、已跳過
- ✅ **即時統計**：顯示已收集、已跳過、剩餘和總數
- ✅ **篩選功能**：支援全部、已收集（含跳過）、未收集三種模式
- ✅ **資料儲存**：自動保存到瀏覽器 localStorage

### 🎨 友善的使用介面
- ✅ **直觀的卡片設計**：視覺上清楚區分三種狀態
- ✅ **快速操作方式**：
  - 點擊卡片標記已收集
  - 右鍵選單標記跳過
  - 一鍵清除所有收集記錄
- ✅ **色違模式切換**：按鈕一鍵切換普通/色違版本精靈圖
- ✅ **RWD 設計**：支援桌面和行動裝置

## 🚀 快速開始

### 使用方式

1. **下載或 clone 此專案**
```bash
git clone https://github.com/yourusername/pokemon_za_list.git
cd pokemon_za_list
```

2. **直接在瀏覽器裡打開 `index.html`**
   - 雙按 `index.html` 檔案，或
   - 在瀏覽器中按 Ctrl+O (Windows) / Cmd+O (Mac)，選擇 `index.html`

3. **開始追蹤色違收集！**

## 📖 使用指南

### 基本操作

| 操作 | 方式 | 說明 |
|------|------|------|
| **標記已收集** | 點擊卡片或勾選框 | 卡片變綠色，顯示 ✓ 符號 |
| **標記跳過** | 右鍵點擊卡片 | 卡片變灰色，顯示 ✕ 符號 |
| **取消跳過** | 右鍵點擊灰色卡片 | 恢復為白色卡片 |
| **切換色違** | 點擊「✨ 色違」按鈕 | 切換精靈圖版本 |
| **篩選寶可夢** | 使用上方篩選下拉菜單 | 全部/已收集/未收集 |
| **清除記錄** | 點擊「清除所有」按鈕 | 清除所有已收集記錄（需確認） |

### 理解統計面板

```
已收集: 154 | 已跳過: 5 | 剩餘: 71 | 總數: 230
```

- **已收集 (154)**：已成功收集的色違寶可夢數量
- **已跳過 (5)**：標記為跳過的寶可夢（如鎖閃神獸）
- **剩餘 (71)**：還需要收集的寶可夢數量 = 230 - 154 - 5
- **總數 (230)**：密阿雷圖鑑總共的寶可夢數量

### 卡片狀態說明

| 狀態 | 外觀 | 含義 |
|------|------|------|
| **未收集** | 白色背景 | 還未收集或取消收集的寶可夢 |
| **已收集** | 綠色背景 + ✓ | 已成功收集的色違寶可夢 |
| **已跳過** | 灰色背景 + ✕ + 刪除線 | 決定不收集的寶可夢（如鎖閃） |

## 💾 資料管理

### local 儲存

所有資料保存在瀏覽器 localStorage 中

```javascript
// localStorage 鍵值對應
'pokemon-collected'   // 已收集的寶可夢 ID 列表
'pokemon-skipped'     // 已跳過的寶可夢 ID 列表
'pokemon-shiny-mode'  // 色違模式開關狀態
```

### 備份和恢復

打開瀏覽器開發者工具（F12）→ 應用程式 → localStorage，可以查看和手動編輯儲存的資料。

## 🎯 篩選邏輯

### 三種篩選模式

- **全部 (230)**
  - 顯示所有 230 個寶可夢
  - 包括已收集、未收集、已跳過

- **已收集**
  - 顯示：已收集 + 已跳過 的寶可夢
  - 隱藏：未收集的寶可夢
  - 用途：查看已完成的目標

- **未收集**
  - 顯示：只有未收集且未跳過的寶可夢
  - 隱藏：已收集和已跳過的寶可夢
  - 用途：專注在還需要收集的目標
